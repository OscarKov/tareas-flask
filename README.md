# tareas-flask
Steps to run the API
1. install or upgrade python to version 3.12.2
2. install pipenv
```bash
  pip install --user pipenv
```
2. clone repository
```bash
  git clone https://github.com/OscarKov/tareas-flask.git
```
3. install dependencies
```bash
  pipenv install
```
4. move to 'Src' directory
```bash
  cd ./Src
```
5. run flask
```bash
  flask run
```
